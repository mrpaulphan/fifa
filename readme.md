# Mangr

## Requirments
- [Node.js](https://nodejs.org/en/)
- Plugins: .editorconfig

## Installation
- `./phan setup`

## Development
- To start a server run `./phan go`
- To start gulp' run `./phan dev`

## Database
After you run `./phan setup`, go to your `.env` file and fill out the "DB" settings.
